# 積み上げのつぶやき

## 概要
その日に勉強する内容をつぶやくアプリ。
<br>
記録に残すことで、過去に何を勉強していて、そして今何を勉強しているかを簡単に振り返ることが出来る。
<br>
どれだけ成長したかもわかるので勉強のモチベーションもあがるかも（？）
